Как React рендерит компоненты в DOM и их изменения?

  • Виртуальный DOM — представление DOM в виде объекта
  • После каждого изменения создается новый VDOM и вычисляется разница между новым и предыдущим
  • Разница двух VDOM примененяется к DOM для его изменения
