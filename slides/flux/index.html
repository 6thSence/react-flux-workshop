Компоненты во Flux

  • Событие — идентификатор изменения + данные
  • Диспетчер — pub/sub для рассылки событий слушателям
  • Хранилище

Как Flux управляет данными и их изменениями?

  • Пользователь нажимает на кнопку
  • В диспетчер отправляется событие
  • Диспетчер уведомляет хранилище
  • Хранилище изменяет состояние и уведомляет компонент
  • Компонент достает данные из хранилища и обновляется
